# ecmascript-2015-iojs
Sandbox for ECMAScript 2015 features available in io.js

## What is supported
* class, extends, static
* let/const
* for..of
* String interpolation (`${name} codes in ${language}`)
* Set, Map
* => (syntax does work, but no automatic binding to this)
* generators
* Promise
* Symbol
